MemoryValue

data class MemoryValue(val value: Long) : Comparable<MemoryValue>

Represents a storage size value in bytes with support for binary units (KiB, MiB, GiB). Provides conversion between different storage units and arithmetic operations.

Transformation to YAML-friendly string is roughed out to the nearest unit. For example, 1024 bytes are represented as "1Ki", 1048576 bytes as "1Mi", and 1073741824 bytes as "1Gi".
