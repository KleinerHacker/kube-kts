class HelmGlobalOptions(var namespace: String? = null, var kubeContext: String? = null, var kubeConfig: String? = null, var kubeApiServer: String? = null, var kubeAsUser: String? = null, var kubeAsGroup: Array<String>? = null, var kubeCaFile: String? = null, var kubeToken: String? = null, var kubeTlsServerName: String? = null, var kubeInsecureSkipTlsVerify: Boolean = false, var burstLimit: Int? = null, var qps: String? = null, var registryConfig: String? = null, var repositoryCache: String? = null, var repositoryConfig: String? = null) : HelmArgsProvider

Global Helm flags that are available to every Helm subcommand and forwarded unchanged to Helm.

This option group is meant to be embedded via picocli @Mixin into all Helm based commands. Only the flags actually provided by the user are translated to Helm arguments in toHelmArgs.
