OwnerReferenceSpecBuilder

class OwnerReferenceSpecBuilder

Builder for creating OwnerReferenceSpec instances.

This builder allows configuration of owner reference properties such as API version, kind, name, and UID. It also supports optional settings like controller flag and block owner deletion behavior.

All values are optional.

Properties

blockOwnerDeletion
Link copied to clipboard
var blockOwnerDeletion: Boolean?

If true, the garbage collector will delete the current object when the owner is deleted.

controller
Link copied to clipboard
var controller: Boolean?

Indicates whether this reference points to a controller that manages the current object. If true, the owner has an exclusive control over the current object and will block its deletion if blockOwnerDeletion is also set to true.
