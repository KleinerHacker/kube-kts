ContainerSecurityContextSpec

constructor(runAsUser: Long?, runAsGroup: Long?, runAsNonRoot: Boolean?, seLinuxOptions: SecurityContextSpec.SELinuxOptionsSpec?, seccompProfile: SecurityContextSpec.ProfileSpec?, appArmorProfile: SecurityContextSpec.ProfileSpec?, windowsOptions: SecurityContextSpec.WindowsOptionsSpec?, privileged: Boolean?, readOnlyRootFilesystem: Boolean?, allowPrivilegeEscalation: Boolean?, procMount: ContainerSecurityContextSpec.ProcMountType?, capabilities: ContainerSecurityContextSpec.CapabilitiesSpec?)

Creates a new instance of the ContainerSecurityContextSpec class.

Parameters

runAsUser

The user ID to run the container process as. Can be null.

runAsGroup

The group ID to run the container process as. Can be null.

runAsNonRoot

Indicates whether the container process should run as a non-root user. Can be null.

seLinuxOptions

SELinux options defining the SELinux context of the container. Can be null.

seccompProfile

Seccomp profile details to enforce syscall policy. Can be null.

appArmorProfile

AppArmor profile details to configure process confinement. Can be null.

windowsOptions

Windows-specific container options for security settings. Can be null.

privileged

Determines whether the container should run in privileged mode. Can be null.

readOnlyRootFilesystem

Indicates if the container's root filesystem should be mounted as read-only. Can be null.

allowPrivilegeEscalation

Indicates if the container process is allowed to gain additional privileges. Can be null.

procMount

Specifies the type of mount configuration for the /proc filesystem. Can be null.

capabilities

Defines the capabilities to add or drop for the container process. Can be null.
