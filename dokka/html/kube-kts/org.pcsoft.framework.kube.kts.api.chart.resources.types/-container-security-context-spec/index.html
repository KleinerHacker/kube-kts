ContainerSecurityContextSpec

class ContainerSecurityContextSpec(runAsUser: Long?, runAsGroup: Long?, runAsNonRoot: Boolean?, seLinuxOptions: SecurityContextSpec.SELinuxOptionsSpec?, seccompProfile: SecurityContextSpec.ProfileSpec?, appArmorProfile: SecurityContextSpec.ProfileSpec?, windowsOptions: SecurityContextSpec.WindowsOptionsSpec?, val privileged: Boolean?, val readOnlyRootFilesystem: Boolean?, val allowPrivilegeEscalation: Boolean?, val procMount: ContainerSecurityContextSpec.ProcMountType?, val capabilities: ContainerSecurityContextSpec.CapabilitiesSpec?) : SecurityContextSpec

Specifies the security context settings for a container within a Kubernetes pod.

This class provides options to configure the security-related attributes of a container, including file system permissions, user and group execution context, privilege levels, and support for SELinux, Seccomp, AppArmor profiles, and Windows-specific settings. The configuration focuses on ensuring containers maintain the necessary security restrictions while allowing for fine-grained control of their behavior.

Parameters

runAsUser

The user ID to run the container process as. Can be null.

runAsGroup

The group ID to run the container process as. Can be null.

runAsNonRoot

Indicates whether the container process should run as a non-root user. Can be null.

seLinuxOptions

SELinux options defining the SELinux context of the container. Can be null.

seccompProfile

Seccomp profile details to enforce syscall policy. Can be null.

appArmorProfile

AppArmor profile details to configure process confinement. Can be null.

windowsOptions

Windows-specific container options for security settings. Can be null.

privileged

Determines whether the container should run in privileged mode. Can be null.

readOnlyRootFilesystem

Indicates if the container's root filesystem should be mounted as read-only. Can be null.

allowPrivilegeEscalation

Indicates if the container process is allowed to gain additional privileges. Can be null.

procMount

Specifies the type of mount configuration for the /proc filesystem. Can be null.

capabilities

Defines the capabilities to add or drop for the container process. Can be null.
