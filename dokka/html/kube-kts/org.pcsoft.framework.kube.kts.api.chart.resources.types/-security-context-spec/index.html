SecurityContextSpec

sealed class SecurityContextSpec

Represents the security context for a container, defining various security-related configurations.

This specification includes settings such as user and group IDs, privilege escalation controls, filesystem permissions, capabilities, SELinux options, seccomp profiles, AppArmor profiles, and Windows-specific options. It is used to enforce security policies at the container level.

Parameters

runAsUser

The user ID to run the container as.

runAsGroup

The group ID to run the container as.

runAsNonRoot

Whether to run the container as a non-root user.

seLinuxOptions

The SELinux options for the container.

seccompProfile

The seccomp profile to apply to the container.

appArmorProfile

The AppArmor profile to apply to the container.

windowsOptions

The Windows-specific options for the container.
