ProfileSpec

data class ProfileSpec(val type: SecurityContextSpec.ProfileType, val localhostProfile: String?)

Specifies the security profile configuration for a pod or container.

This data class defines the type of security profile to be applied and, if applicable, the localhost profile string. It enforces validation rules to ensure that when the type is set to ProfileType.Localhost, the localhostProfile must not be null or blank.
