VolumeClaimTemplateSpec

data class VolumeClaimTemplateSpec(val metadata: VolumeClaimTemplateSpec.Metadata, val spec: VolumeClaimTemplateSpec.Spec)

Represents a PersistentVolumeClaim template used by a Kubernetes StatefulSet.

A StatefulSet provisions one PersistentVolumeClaim per Pod for each entry of its volumeClaimTemplates. The created claims provide stable, per-Pod storage that survives Pod rescheduling.
