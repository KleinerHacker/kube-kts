PodSecurityContextSpecBuilder

class PodSecurityContextSpecBuilder : SecurityContextSpecBuilder<PodSecurityContextSpec>

A builder class for constructing a PodSecurityContextSpec instance.

This builder provides methods to configure the various attributes of a pod's security context. Common attributes include file system group settings, supplemental group policies, and system-level configurations like sysctls.

The PodSecurityContextSpecBuilder allows for precise customization and is best suited for scenarios where programmatic creation of security context specifications is required.

Constructors for this class are internal, and it inherits from SecurityContextSpecBuilder to retain additional shared configuration behaviors.
