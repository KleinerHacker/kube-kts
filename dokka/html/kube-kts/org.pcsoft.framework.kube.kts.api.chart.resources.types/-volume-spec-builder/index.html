VolumeSpecBuilder

class VolumeSpecBuilder

A builder class for constructing volume specifications to be used in a Kubernetes pod.

This builder provides methods for configuring various types of volume sources, such as ConfigMaps, Secrets, and HostPaths. The constructed volume specification can be integrated into Kubernetes pod configurations.

Some values are required.

Parameters

name

The name of the volume being constructed.
