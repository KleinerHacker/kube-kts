ContainerSecurityContextSpecBuilder

class ContainerSecurityContextSpecBuilder : SecurityContextSpecBuilder<ContainerSecurityContextSpec>

Builder class for constructing a ContainerSecurityContextSpec object which defines the security-related configurations for a container.

This class allows setting specific properties related to the container's security context, such as process namespace configuration (procMount) and defining capabilities to be added or dropped. It extends from SecurityContextSpecBuilder to inherit common security context configuration options.

This builder is not intended for direct instantiation but is created and utilized through internal mechanisms.
