class NodeAffinitySpecBuilder

Builder class to configure node affinity specifications for scheduling and execution of workloads.

Node affinity rules can specify requirements for node selection during scheduling, as well as preferences that influence scheduling decisions. This builder allows defining both required and preferred rules with flexible configurations.

The required rules enforce strict constraints that nodes must satisfy, while preferred rules specify a weighted preference towards certain nodes.

Internal constructor is used to ensure controlled instantiation, typically as part of a larger configuration system.

Functions provide the ability to add individual terms for required and preferred rules, as well as configure lists of terms using nested builders.
