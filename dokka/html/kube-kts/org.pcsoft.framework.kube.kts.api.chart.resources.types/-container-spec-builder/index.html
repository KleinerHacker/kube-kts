ContainerSpecBuilder

class ContainerSpecBuilder

Builder class for defining and configuring container specifications.

The ContainerSpecBuilder is used to create and configure container definitions in a fluent manner. It provides methods to define container attributes such as ports, environment variables, resource limits, volume mounts, lifecycle configurations, and security contexts.

This builder is intended to construct specifications for containerized applications, including their associated runtime configurations and deployment behaviors.
