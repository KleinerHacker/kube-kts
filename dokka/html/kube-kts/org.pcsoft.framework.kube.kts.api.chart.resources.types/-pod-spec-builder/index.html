PodSpecBuilder

class PodSpecBuilder

Builder class for configuring and constructing the specification of a Kubernetes Pod.

The PodSpecBuilder class provides an extensive set of methods and properties to define the configuration of a Kubernetes Pod, including containers, networking settings, volumes, security, resource claims, and more.
