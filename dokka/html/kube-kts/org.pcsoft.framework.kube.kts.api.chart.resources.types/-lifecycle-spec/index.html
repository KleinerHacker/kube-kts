LifecycleSpec

data class LifecycleSpec(val postStart: LifecycleSpec.Action?, val preStop: LifecycleSpec.Action?)

Defines the lifecycle configuration for handling container events within a Kubernetes pod.

This class specifies custom actions that can be executed at specific points in a container's lifecycle, such as immediately after startup (postStart) or just before termination (preStop).
