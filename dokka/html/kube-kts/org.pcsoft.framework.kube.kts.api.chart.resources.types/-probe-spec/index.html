ProbeSpec

data class ProbeSpec(val action: ProbeSpec.ProbeAction, val initialDelaySeconds: Duration?, val periodSeconds: Duration?, val timeoutSeconds: Duration?, val successThreshold: Int?, val failureThreshold: Int?, val terminationGracePeriodSeconds: Duration?)

Represents the configuration of a probe, which is used to check the health or readiness of a container or a service in a Kubernetes environment. Probes can be configured with different actions and thresholds to determine success or failure based on specific criteria.
