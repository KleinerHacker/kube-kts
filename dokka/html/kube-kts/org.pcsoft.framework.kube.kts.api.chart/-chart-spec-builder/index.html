ChartSpecBuilder

class ChartSpecBuilder

A DSL builder for constructing ChartSpec objects.

This class provides a fluent API for configuring Helm chart metadata, including dependencies, maintainers, keywords, and Kubernetes version compatibility. It is typically used via the top-level chart function.

All values are optional.

Parameters

name

The unique name of the Helm chart.

version

The semantic version of the chart.
