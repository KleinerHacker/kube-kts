ValueAccess

class ValueAccess

The ValueAccess class provides utility functions to encapsulate access to nodes within a JSON structure. It supports typed retrieval of values, iteration over arrays and maps, and traversal of nested objects.

This class is designed to work with JSON structures and exposes a flexible API for accessing and manipulating data in a type-safe manner.

Parameters

node

The JSON node instance that represents the underlying JSON structure.

basePath

Optional base path used for key resolution in nested JSON structures.
