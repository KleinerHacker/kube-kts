ExplicitTemplateSpecBuilder

class ExplicitTemplateSpecBuilder<S : ResourceSpec, B : ResourceSpecBuilder<S>> : TemplateSpecBuilder<S, ExplicitTemplateSpec<S>>

Builder class for creating instances of ExplicitTemplateSpec. This class provides methods to configure the metadata and specification of a Kubernetes resource template while ensuring that all necessary components are set before the final build.

Some values are required.

Parameters

apiVersion

The API version of the resource being configured.

kind

The kind of the resource being configured (e.g., ConfigMap, Pod, Service).

specBuilder

An instance of the builder responsible for creating the resource specification.

Type Parameters

S

The type of the resource specification contained within the template. Must extend ResourceSpec.

B

The type of the builder used to configure the resource specification. Must implement ResourceSpecBuilder.
