BaseDirectHelmCommand

sealed class BaseDirectHelmCommand : BaseCommand, Callable<Int> , HelmCommandLineProvider

Base class for Helm commands that operate on an already installed release (e.g. status, list, history).

Unlike BaseRenderedHelmCommand, this class does not derive from BaseRenderCommand: it neither requires a REPOSITORY positional nor runs the Scan → Compile → Render pipeline. The KTS scripts are irrelevant for these commands, so the assembled command line is forwarded directly to helm.

The call() boilerplate (logging setup, flag validation, exception handling) is intentionally duplicated from BaseRootCommand instead of inherited, because BaseRootCommand enforces a mandatory REPOSITORY positional that release commands must not have.

Subclasses define the Helm subcommand and positionals via helmCommand and contribute their option groups via helmOptionGroups.
