Type

enum Type : Enum<ServiceSpec.Type>

Represents the type of service in Kubernetes.

The Type enum defines the ways a Kubernetes service can expose itself to the network. Each value corresponds to a specific service exposure method.

This enum is used by ServiceSpec to specify the desired service type and its behavior within the cluster or externally.
