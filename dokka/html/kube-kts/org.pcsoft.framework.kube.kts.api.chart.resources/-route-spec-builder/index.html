RouteSpecBuilder

class RouteSpecBuilder : ResourceSpecBuilder<RouteSpec>

Builder class for constructing instances of RouteSpec, the specification for an OpenShift Route.

Note: Route is an OpenShift/OKD specific resource and is not available on vanilla Kubernetes clusters. On standard Kubernetes use the ingress { } DSL instead.

The builder allows configuration of the external host and path, the primary backend (to), additional weighted backends (addAlternateBackend / alternateBackends), the target port, TLS termination (tls) and the wildcardPolicy.

All values are optional.
