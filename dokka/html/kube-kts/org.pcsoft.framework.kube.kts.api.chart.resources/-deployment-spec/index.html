DeploymentSpec

data class DeploymentSpec(val replicas: Int?, val selector: LabelSelectorSpec, val template: PodTemplateSpec, val strategy: DeploymentStrategySpec?, val minReadySeconds: Duration?, val revisionHistoryLimit: Int?, val paused: Boolean?, val progressDeadlineSeconds: Duration?) : ResourceSpec

Represents the specification for a Kubernetes Deployment resource.

A DeploymentSpec defines the desired state for a Deployment, which is a higher-level abstraction for managing Pods. Deployments provide declarative updates for Pods and ReplicaSets while offering features such as automated rollouts, rollbacks, scaling, and more.
